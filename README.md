# Docs2Pdf

Aplicação web para conversão de arquivos Office e imagens em PDF.

- `frontend/`: frontend Angular 17
- `backend/`: backend .NET 8 Web API

## Como começar

### Frontend

```bash
cd frontend
npm install
npm run start
```

### Backend

```bash
cd backend
dotnet run
```

## Observação
Não há testes automatizados neste projeto; qualidade é garantida por revisão e validação manual.
